# Run and deploy ApexBlueprintV4

This repository runs a Node/Vite app with optional provider integrations.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env` and configure `VEKLOM_API_KEY` and other provider settings as needed.
3. Run the app:
   `npm run dev`
